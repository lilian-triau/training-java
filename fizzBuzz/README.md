# FizzBuzz

Lister les chiffres de 1 à 100
Retourner le chiffre tel quel sauf dans les cas particuliers suivants :
* quand le chiffre est un multiple de trois : retourner fizz
* quand le chiffre est un multiple de 5 : retourner buzz
* quand le chiffre est un multiple de 3 et de cinq : retourner fizzbuzz  
 
## Focus technique

Aucune contrainte de framework, de librairie ou de méthode
Il faut juste pouvoir prouver que cela fonctionne.
